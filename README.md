# 📊 Amazon Sales Analysis PowerBI Dashboard

## 📌 Brief One Line Summary
An interactive Business Intelligence dashboard built using Power BI to analyze Amazon sales performance, customer behavior, product profitability, and shipping operations.

---

# 🔍 Overview
This project is a complete end-to-end Business Intelligence solution developed using Power BI. The dashboard transforms raw Amazon transactional data into meaningful business insights through interactive visualizations, KPI analysis, operational tracking, and advanced reporting.

The project focuses on analyzing:
- Sales performance
- Product profitability
- Customer behavior
- Shipping efficiency
- Operational performance

The dashboard was designed with an executive-level reporting approach to support data-driven business decisions.

---

# ❗ Problem Statement
Organizations often struggle to analyze large volumes of sales and operational data efficiently. Raw datasets make it difficult to identify profitable products, customer trends, operational delays, and revenue-driving business segments.

This project solves that challenge by creating a centralized interactive dashboard that helps management:
- Monitor business growth
- Track profitability
- Identify operational bottlenecks
- Analyze customer contribution
- Improve strategic decision-making

---

# 📂 Dataset
The dataset contains Amazon sales transactional data including:

- Order Details

# ⚙️ Methods & Workflow

## 📌 Data Cleaning & Transformation
- Removed null and duplicate values
- Corrected data types
- Created custom columns
- Performed data transformation using Power Query

## 📌 Data Modeling
- Built relationships between tables
- Created a custom calendar table using `List.Dates()`
- Optimized the data model for reporting

## 📌 DAX & KPI Development
Created advanced business measures using DAX including:
- Total Sales
- Total Profit
- Profit Margin %
- Customer Retention %
- On-Time Delivery %
- Avg Shipping Days
- Late Deliveries
- Product Profitability Measures

## 📌 Dashboard Design
- Multi-page dashboard architecture
- Interactive slicers & filters
- KPI cards
- Drill-down analysis
- Executive-level visual storytelling

# 📈 Key Business Insights

- Technology category generated the highest revenue contribution
- Some products showed high sales but low profitability
- Heavy discounts negatively impacted profit margins
- Consumer segment contributed the highest sales share
- Standard Class shipping had the longest delivery duration
- Operational delays were identified in certain regions
- Repeat customers significantly contributed to total revenue
